# Plan B（后手）— 逻辑与功能性说明

本文档描述应用的界面结构、用户操作与结果（谁在哪儿点了什么、会打开什么、发生什么）。

---

## 一、应用入口与主结构

- **启动应用** → 进入根视图 `ContentView`，其仅承载 **主 Tab 栏**（`MainTabView`）。
- **主 Tab 栏** 固定四个 Tab：**首页**、**线索库**、**发送**、**设置**。用户点击 Tab 切换页面，当前选中的 Tab 高亮显示。

---

## 二、首页（Home）

| 用户操作 | 结果 |
|----------|------|
| 进入首页 | 看到标题、发送计划状态徽标（已开启/已暂停）、倒计时大卡片（距预计发送日期的天数 + 时分秒）、预计发送日期、快捷入口卡片。 |
| 点击倒计时区域下方的「刷新倒计时」按钮 | 触觉反馈；预计日期按「当前时间 + 发送页当前触发周期天数」重新计算并写入，倒计时大卡片随之更新。不点击该按钮时，预计日期与倒计时不会因发送页修改触发周期而变化。 |
| 点击「线索库」快捷入口卡片 | 触觉反馈；Tab 自动切换到 **线索库**。 |
| 点击「发送设置」快捷入口卡片 | 触觉反馈；Tab 自动切换到 **发送**。 |

- 预计日期仅在用户点击「刷新倒计时」时计算：预计日期 = 当前时间 +「发送」页触发周期天数；倒计时大卡片 = 预计日期时间 − 当前时间（天数 + 时分秒）。倒计时保持每秒更新（时分秒走秒）。若已开启计划但从未点击刷新，预计日期显示「点击下方「刷新倒计时」计算」。

---

## 三、线索库（Clues）

| 用户操作 | 结果 |
|----------|------|
| 进入线索库 | 看到标题、搜索框、分段筛选（全部 / 重要物品 / 数字资产 / 给家人的话）、隐私提示条、导出入口卡片、线索列表（或空状态）。筛选标签可拖拽调整顺序。 |
| 在搜索框输入文字 | 列表仅显示标题或各字段内容包含该关键词的线索；点击搜索框内清空按钮可清除搜索。 |
| 点击「全部」或某一类型筛选 | 列表仅显示该类型或全部线索；选中态高亮。 |
| 拖拽类型标签 | 筛选标签顺序改变，列表排序按新顺序（类型优先，同类型按更新时间倒序）。 |
| 点击右上角「+」按钮 | 弹出 **添加线索** Sheet（`AddClueSheet`），可输入新线索并保存。 |
| 点击「导出线索手册」卡片 | 弹出 **导出 PDF** Sheet（`ExportPDFSheet`），可设置文件名、密码并导出。 |
| 点击某条线索行 | 弹出 **线索详情** Sheet（`ClueDetailSheet`），可查看详情、编辑或删除。 |
| 在某条线索上长按/右键「删除」 | 从列表中删除该线索（需在详情内确认删除的流程见下文）。 |

---

## 四、添加线索（AddClueSheet）

| 用户操作 | 结果 |
|----------|------|
| 打开添加 Sheet | 顶部为类型选择（重要物品 / 数字资产 / 给家人的话），下方为当前类型对应的表单字段。 |
| 切换类型 | 表单切换为该类型的输入项：重要物品（名称、存放位置、关联、备注）；数字资产（平台、账号、关联、备注）；给家人的话（标题、对象、背景、正文）。 |
| 填写后点击「完成」 | 若必填项已填（重要物品：名称；数字资产：平台；给家人的话：标题），则保存新线索并关闭 Sheet；否则「完成」不可点。 |
| 点击「取消」 | 不保存，关闭 Sheet。 |

- 从线索库打开时，可传入当前选中的筛选类型作为初始类型。

---

## 五、线索详情（ClueDetailSheet）

| 用户操作 | 结果 |
|----------|------|
| 打开详情 Sheet | 显示该线索的类型标签、全部字段内容、更新时间、删除按钮。 |
| 点击导航栏「编辑」 | 弹出 **编辑线索** Sheet（`EditClueSheet`），可修改各字段。 |
| 在编辑 Sheet 中保存 | 详情中的内容更新，编辑 Sheet 关闭。 |
| 点击「完成」 | 关闭详情 Sheet。 |
| 点击「删除」按钮 | 弹出确认对话框「删除这条线索？」；确认后删除该线索并关闭详情 Sheet。 |

---

## 六、编辑线索（EditClueSheet）

| 用户操作 | 结果 |
|----------|------|
| 打开编辑 Sheet | 显示当前线索类型（不可改）及该类型下的可编辑字段，预填原有内容。 |
| 修改字段后点击「完成」 | 若必填项仍满足，则更新线索并回调刷新详情，关闭 Sheet。 |
| 点击「取消」 | 不保存修改，关闭 Sheet。 |

---

## 七、导出线索手册（ExportPDFSheet）

| 用户操作 | 结果 |
|----------|------|
| 打开导出 Sheet | 可设置文件名、是否使用密码、密码与确认、密码提示；有预览信息与导出按钮。 |
| 勾选使用密码并填写密码/确认/提示 | 仅当两次密码一致且密码非空时，才允许点击导出。 |
| 点击「导出」 | 若无线索则不可导出；否则生成 PDF（含封面、按类型分节、每条线索一页、尾页），若设置了密码则加密；完成后弹出系统分享面板，可保存或分享文件。 |
| 点击「取消」 | 关闭 Sheet。 |

---

## 八、发送（Send）

| 用户操作 | 结果 |
|----------|------|
| 进入发送页 | 看到标题、状态徽标（已开启/已暂停）、收件联系人区（邮箱/手机输入与列表）、展示姓名、通知正文（可展开查看预览）、触发周期滑块（7–365 天）、预计发送日期、开启/关闭计划主按钮、底部说明文字。 |
| 输入邮箱并点击「+」 | 将该邮箱加入收件列表，输入框清空。 |
| 输入手机号并点击「+」 | 将该手机号加入收件列表，输入框清空。 |
| 点击某条邮箱/手机旁的删除图标 | 从收件列表中移除该邮箱或手机号。 |
| 修改「展示姓名」 | 通知正文预览中的称呼与展示名同步更新。 |
| 点击「通知正文」区域 | 展开/收起正文与「查找线索」指引的编辑与预览。 |
| 编辑「查找线索」指引文案 | 仅该段内容可编辑，用于在通知中提示收件人如何开始查找。 |
| 拖动「触发周期」滑块 | 天数变化，预计发送日期随之重新计算并显示。 |
| 点击「开启发送计划」按钮 | 计划开启，sendDate 按当前天数从当前时间起算；状态徽标变为「已开启」，按钮变为「关闭发送计划」。 |
| 点击「关闭发送计划」按钮 | 计划关闭，状态徽标变为「已暂停」，按钮变为「开启发送计划」。 |

## 九、设置（Settings）

| 用户操作 | 结果 |
|----------|------|
| 进入设置页 | 看到标题、账户卡、通用设置卡片、重置按钮、底部 Slogan。 |
| **未登录时** 点击账户卡「登录/注册」区域 | 弹出 **登录/注册** Sheet（`LoginRegisterSheet`），可输入账号密码进行登录或注册。 |
| **已登录时** 查看账户卡 | 显示头像占位、昵称或邮箱、邮箱；右侧有「退出」按钮。 |
| 点击「退出」 | 弹出确认对话框；确认后清除本地登录状态，账户卡恢复为「登录/注册」入口。 |
| 点击「主题风格」行 | 弹出 **主题选择** Sheet（`ThemePickerSheet`），选择后应用主题立即生效，关闭 Sheet。 |
| 点击「语言」行 | 弹出 **语言选择** Sheet（`LanguagePickerSheet`），选择后应用语言立即生效，约 0.3 秒后自动关闭 Sheet。 |
| 「FaceID 安全锁定」 | 当前为占位，显示「PRO」。 |
| 「使用指南 & 常见问题」/「关于后手」 | 当前为占位，仅展示入口。 |
| 点击「重置并删除本地数据」 | 弹出确认对话框；确认后执行 `appStore.resetAll()`（清空线索与发送计划等本地数据），对话框关闭。 |

## 十、登录/注册（LoginRegisterSheet）

| 用户操作 | 结果 |
|----------|------|
| 打开 Sheet | 顶部为模式切换：**登录** / **注册**；下方为邮箱、密码等输入项。 |
| 选择「登录」 | 显示邮箱、密码；点击「登录」时校验邮箱格式、本地是否已有该邮箱及密码是否正确；成功则写入 Keychain 并更新为已登录，关闭 Sheet。 |
| 选择「注册」 | 显示邮箱、密码、确认密码、昵称（选填）；点击「注册」时校验邮箱格式、密码长度≥6、两次密码一致；成功则创建本地账户并登录，关闭 Sheet。 |
| 校验失败 | 在表单上方显示错误文案（如「邮箱格式无效」「密码过短」等），不关闭 Sheet。 |
| 点击「取消」 | 关闭 Sheet，不保存。 |

## 十三、数据与状态概要

- **线索列表与发送计划**：由 `AppStore` 管理，变更时通过 `JSONStore` 持久化到 Documents（`clues.json`、`plan.json`）。  
- **登录状态与当前用户**：由 `AuthService` 管理，凭证与昵称等存 Keychain；`SettingsView` 与 `LoginRegisterSheet` 依赖其 `isLoggedIn`、`currentUser`。  
- **主题与语言**：由 `ThemeManager`、`LocalizationManager` 单例管理，设置页的入口仅打开对应 Sheet 并应用选择结果。  
- **本地通知**：由 `NotificationService` 根据发送计划的 `sendDate` 与 `intervalDays` 在 7 天/3 天/24 小时前安排提醒；阶段 A 不真实发送邮件/短信。

## 十四、界面层级与弹窗关系简图

```
MainTabView
├── HomeView
├── CluesView
│   ├── Sheet: AddClueSheet
│   ├── Sheet: ExportPDFSheet
│   └── Sheet: ClueDetailSheet
│       └── Sheet: EditClueSheet
├── SendView
└── SettingsView
    ├── Sheet: LoginRegisterSheet
    ├── Sheet: ThemePickerSheet
    └── Sheet: LanguagePickerSheet
```

- 所有 Sheet 均可通过「取消」或导航栏关闭/下滑关闭；登录/注册、编辑、导出等会在完成或确认后自动关闭。
